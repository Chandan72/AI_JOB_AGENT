## Usage Notes
- **Replace bracketed placeholders** like `[Recruiter Name]`, `[Company]`, and `[specific industry/tech area]` with personalized details. If no specific info is available, use generic but relevant terms like "your space" or "your platform."
- **Version A** targets the recruiter's priority: filling roles efficiently with proven talent. It highlights pipeline-ready value (efficiency gains, scalability).
- **Version B** targets the hiring manager's priority: technical impact and problem-solving. It leads with a concrete technical achievement (30% faster processing) relevant to engineering leadership.
- **Send Version A first** via LinkedIn InMail or email. Wait 5 business days before sending Version B if no reply. Never send both on the same day.
- **Proofread** to ensure the candidate's actual achievements (e.g., "40% efficiency gains") match their profile bullets exactly.
